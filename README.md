LWC Modal
==

## Intro
쉬운 modal 구현

## Attribute
|Value|Type|Default|Description|
|---|---|---|---|
|`width`|number|50%|px|
|`hideClose`|boolean|false||
|`hideHeader`|boolean|false||
|`hideFooter`|boolean|false||

<br>

## Method
|Name|Description|
|---|---|
|`open()`|Open the modal|
|`close()`|Close the modal|

<br>
